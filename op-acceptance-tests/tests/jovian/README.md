On Sepolia, the Jovian L2 hardfork activates after the BPO2 L1 hardfork (i.e., activates against a BPO2 L1).
On mainnet, the Jovian L2 hardfork activates before the Fusaka L1 hardfork (i.e., activates against a Pectra L1).

We test the Jovian changes against both a Fusaka L1 and a Pectra L1 in the `bpo2` and `pectra` subpackages.
